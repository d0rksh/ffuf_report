# Fuff Report

## Installation
1. git clone https://github.com/ffuf/ffuf.git
2. cd ffuf/pkg/output
3. open file_html.go file in code editor
4. go to line 20 ```const htmlTemplate = "paste_here"```
5. copy the template.tp and replace ```const htmlTemplate```
